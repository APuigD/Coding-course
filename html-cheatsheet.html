Lists

You can create 2 types of lists Unordered lists and Ordered lists :
An unordered list is done with a "ul" tag and looks like this:

  • Un
  • Dos
  • Tres

An ordered list is done with a "ol" tag and looks like this:

  1. Un
  2. Dos
  3. Tres

Images

This is some of the fun stuff to do, such as adding images, which you do with the "img" tag. Note that this tag doesn't need a closing tag!

What another one? Here you go:

To do this I needed to add an attribute into the "img" tag: ' src="..." ' . This can be linked to an image with a full url, or an image from the same folder ie ./image.png

Very important!
Remember to add the "alt" attribute so it displays text if the image is not supported.
